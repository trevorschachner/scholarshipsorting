# Scholarship Sorting

Combine pdf files into one file for scholarship grading

## Dependecies
- Pandas: `pip install pandas`
- fitz: `pip install PyMuPDF`
- Python 3


## Usage 

```
# check python version
python --version
python3 --version

# run with Python 3
python scholarshipsorting.py
# or
python3 scholarshipsorting.py
```
